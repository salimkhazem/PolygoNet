## Dependencies 
For the different parts you can install required dependecies by running the following command: 

### For MATC
  #### Install in local 
```bash 
  bash install_MATC.sh
````
  #### Use Docker 
```bash
  docker build -t matc .
```
### For Deep Learning pipeline 
  ```bash
  pip install -r requirements.txt 
  ````
